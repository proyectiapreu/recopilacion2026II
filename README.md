## Documento de consulta

El archivo `data/respuestas_estudiantes.csv` debe tener una columna `documento` y las columnas `P1` a `P120`.

## Escala implementada

La página calcula aciertos por área y los transforma con una escala estandarizada de referencia: puntajes de área con media aproximada 10 y desviación estándar 1, y puntaje global con media aproximada 500 y desviación estándar 100. El archivo `referencia.csv` contiene 1000 participantes de referencia generados con una simulación tipo Rasch simplificada a partir de dificultad de ítems y habilidad por área.

Este reporte es una estimación pedagógica para orientar la preparación. No corresponde a un resultado oficial de admisión.
